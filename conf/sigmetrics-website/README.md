# SIGMETRICS Solar-style Site (Astro)

## Run locally
```bash
npm i
npm run dev
```

## Edit content
- Navigation & site metadata: `src/content/site.ts`
- Officers & committees: `src/content/officers.ts`
- Pages live under `src/pages/` and are root-level.

## Replace branding
- Logo: `public/images/logo.svg`
- Hero image: `public/images/hero.jpg`
