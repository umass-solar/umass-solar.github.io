# SIGMETRICS Solar-style Site (Astro)

This zip contains a modular Astro project that recreates the SIGMETRICS root pages using a SOLAR-like template.

## Requirements
- Node.js + npm (or pnpm/yarn)

## Run locally
```bash
npm i
npm run dev
```

## Where to edit
- Site metadata + nav: `src/content/site.ts`
- Pages: `src/pages/*`
- Components: `src/components/*`
- Global styles: `src/styles/global.css`

## Assets
- `public/assets/acm.gif` (downloaded from the SIGMETRICS site) + a converted `acm.png`
